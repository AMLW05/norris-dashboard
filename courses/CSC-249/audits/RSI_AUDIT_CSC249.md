## What is RSI Alignment?

The RSI framework ensures students experience a coherent course where:
- **Syllabus/Outcomes** = The promise (what you'll learn)
- **Instruction** = The practice (how you'll learn it)
- **Rubrics** = The proof (how you'll demonstrate it)

**Misalignment creates**:
- Student confusion ("Why are we graded on this? It wasn't in the outcomes!")
- Wasted learning time (teaching skills not assessed)
- Validity issues (assessing skills not taught)

**Alignment creates**:
- Clear expectations
- Efficient learning
- Valid assessment
- Student success
