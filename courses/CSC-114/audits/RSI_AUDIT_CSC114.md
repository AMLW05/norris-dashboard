## What is RSI Alignment?

The RSI framework ensures students experience a coherent course where:
- **Syllabus/Outcomes** = The promise (what you'll learn)
- **Instruction** = The practice (how you'll learn it)
- **Rubrics** = The proof (how you'll demonstrate it)

**Misalignment creates**:
- Student confusion ("Why are we graded on this? It wasn't in the outcomes!")
- Wasted learning time (teaching skills not assessed)
- Validity issues (assessing skills not taught)

**Alignment creates**:
- Clear expectations
- Efficient learning
- Valid assessment
- Student success

## Audit Findings Summary

### Strengths (Exemplary Alignment)

1. **100% CLO Coverage**: Every outcome is taught AND assessed
2. **Consistent Weighting**: 70/30 (weekly) to 50/50 (sprints) progression matches PM skill development
3. **Progressive Complexity**: Rubrics evolve from basic to advanced criteria
4. **No Orphan Content**: Nothing taught that isn't assessed; nothing assessed that isn't taught
5. **Student Coherence**: Clear expectations throughout

### Minor Gaps

1. **Communication Skills**: Presentation skills assessed (15 pts) but not explicit in CLOs
   - **Impact**: Low (clearly necessary for professional practice)
   - **Recommendation**: Add CLO-I2: "Communicate technical and process decisions professionally"

2. **Integration CLO Underweighted**: CLO-I1 only explicitly assessed in presentation (15/700 pts)
   - **Impact**: Low (integration is assessed implicitly throughout)
   - **Recommendation**: Accept current structure OR add integration rubric criteria to capstone

3. **Module Learning Outcomes**: No explicit MLOs in assignment files
   - **Impact**: Medium (makes weekly focus less clear)
   - **Recommendation**: Add MLO section to first assignment of each module
