## Usage Notes for Instructors

### First Time Teaching CSC-114
1. Review all assignments in order
2. Read PM mini-lessons (they integrate into lectures)
3. Set up GitHub organization for student repos
4. Prepare sprint board examples
5. Practice sprint demos

### Time Investment
- **Prep**: ~2 hours/week (lecture + PM mini-lesson)
- **Grading**: ~1 hour/student/sprint (but gets faster with practice)
- **Office Hours**: Expect more traffic during first 2 sprints

### Common Student Challenges
1. **Week 1-2**: Environment setup issues
2. **Week 3-4**: Understanding story points
3. **Week 5-6**: First sprint feels overwhelming
4. **Week 7-8**: Sprint fatigue (encourage through it!)
5. **Week 9+**: PM becomes automatic (smooth sailing)
